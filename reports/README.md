# Reports 报告目录

本目录用于存放项目报告和分析结果。

## 报告类型

- **回测报告**: 策略回测结果和性能指标
- **研究报告**: 因子分析、模型研究等
- **可视化图表**: 图表、图像等可视化结果

## 组织结构

```
reports/
├── backtest/          # 回测报告
├── research/          # 研究报告
└── figures/           # 图表和可视化
```

## 报告格式

推荐使用以下格式:
- Markdown (.md)
- PDF (.pdf)
- HTML (.html)
