# Quant Lab - 量化交易研究实验室

一个结构化的量化交易研究平台，旨在将「玩票式脚本」升级成「可迭代的研究工程」。

## 📁 项目结构

```text
quant-lab/
├── data/              # 原始数据 & 处理后的数据
│   └── sample_stock_data.csv    # 示例股票数据
├── notebooks/         # 实验用 Jupyter Notebook
├── backtest/          # 回测引擎 & 策略类
├── reports/           # 各项目报告和分析结果
├── config/            # 全局配置，例如数据库、数据源等
├── demo.py            # 最小可运行演示
└── requirements.txt   # Python 依赖列表
```

## 🎯 项目目标

- **结构化研究**: 提供统一的项目结构，避免代码和数据分散
- **可复现性**: 确保研究结果可以被复现
- **模块化**: 支持独立的策略开发和回测
- **持续迭代**: 为后续所有量化项目提供共同容器

## 📚 使用指南

### 添加新数据
1. 将数据文件放在 `data/` 目录下
2. 推荐使用 CSV 格式，便于读取和版本控制
3. 大文件建议使用 `.gitignore` 排除，避免提交到仓库

### 开发新策略
1. 在 `backtest/` 目录下创建策略类
2. 在 `notebooks/` 中进行实验和可视化
3. 将最终结果保存到 `reports/` 目录

### 配置管理
- 数据库连接、API 密钥等配置放在 `config/` 目录
- 敏感信息使用环境变量或配置文件（记得添加到 `.gitignore`）
