# Notebooks 笔记本目录

本目录用于存放 Jupyter Notebook 实验文件。

## 使用指南

### 启动 Jupyter

```bash
jupyter notebook
```

### Notebook 命名规范

建议使用以下命名格式:
- `01_data_exploration.ipynb` - 数据探索
- `02_feature_engineering.ipynb` - 特征工程
- `03_model_training.ipynb` - 模型训练
- `04_backtest_analysis.ipynb` - 回测分析

## 最佳实践

1. 每个 notebook 应该有清晰的目标和说明
2. 使用 Markdown 单元格记录思路和发现
3. 关键结果保存到 `reports/` 目录
4. 清理输出后再提交到 Git（避免文件过大）
